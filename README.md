## Admin login route

Open `/admin-login` first, enter the admin credentials from `.env`, then you will be redirected to `/admin`.

Local:
`http://localhost:5001/admin-login`

Live:
`https://your-live-url/admin-login`

## MongoDB setup

Copy `.env.example` to `.env` and set your real `MONGO_URI`, `ADMIN_USERNAME`, and `ADMIN_PASSWORD`.
